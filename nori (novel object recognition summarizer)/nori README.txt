Novel Object Recognition data Interpreter
(A simple script to summarize novel object recognition result)


by Abdullah Syafiq, 2022

=====================================================
Instruction:

1. Download python 3.9 (this script was only tested on python 3.9): 
https://www.python.org/ftp/python/3.9.10/python-3.9.10-amd64.exe
2. Install python 3.9. Check "Add Python 3.9 to PATH". Select disable path length limit.

3. Create sample_list.csv. Fill your animal_id on the first column. 
WARNING!! (animal_id has to be exactly the same with the one you input in the machine). 
4.Fill novel object location (1 or 2) on the second column. Please see the example sample_list.csv

5. Please put the script and sample_list.csv on the top of the directory where all your result file exist.
6. Run the script by double clicking it.
7. Summary files will be on output folder.
